# 🍕 Pizza Sales Dashboard

## 📊 Overview
This project is an interactive **Power BI dashboard** built to analyze pizza sales data.  
It provides insights into **revenue, orders, pizza categories, sizes, and sales trends**.

## 📌 Key Metrics
- **Total Revenue:** 817.86K
- **Average Order Value:** 38.31
- **Total Pizzas Sold:** 50K
- **Total Orders:** 21K
- **Avg Pizzas per Order:** 2.32

## 📈 Insights
- **Classic pizzas** have the highest sales.
- **Large size pizzas** generate the most revenue.
- **Friday and weekends** show higher order volumes.
- Top pizzas include **Thai Chicken, Barbecue, California, and Classic Deluxe**.

## 🎛 Features
- Pizza category filter
- Order date range filter
- Monthly order trend
- Revenue by size
- Sales by category
- Top pizza performance

## 🛠 Tools Used
- Power BI  
- Excel / CSV Dataset  
- Data Visualization
